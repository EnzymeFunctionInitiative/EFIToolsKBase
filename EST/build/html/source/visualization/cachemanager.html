class CacheManager(cache_dir: str, max_filehandles=20, filehandle_dump_size=10)

An on-disk append-only dictionary

CacheManager uses the filesystem to store lists of values associated with a key. Every key added to the cache gets its own file. A limited number of file handles are maintained on a least-recently-used basis.

Example

CacheManager can be used as a context manager in the following way

with CacheManager(cachedir) as cm:
    for batch in pf.iter_batches():
        for line in batch.to_pylist():
            cm.append(line["alignment_score"], line["alignment_length"], line["pident"])
